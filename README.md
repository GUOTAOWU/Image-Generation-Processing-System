プロジェクト概要：AI画像処理・生成Webアプリケーション
このプロジェクトは、FastAPI（バックエンド）とStreamlit（フロントエンド）を組み合わせた、多機能なAI画像処理および画像生成プラットフォームです。
ユーザーはウェブブラウザを通じて、画像の文字認識（OCR）、背景除去、編集、そしてAIによる画像生成を行うことができます。また、ユーザー認証システムを備えており、ログインしたユーザーのみが機能を利用できる設計になっています。
主な機能
1.	AI画像生成（Text-to-Image）
機能: テキストプロンプト（指示文）を入力することで、AIが画像を生成します。
技術: Alibaba Cloudの「Dashscope (Qwen-image)」APIを使用しており、プロンプトの自動拡張や透かしの有無なども設定可能です。生成された画像はZIP形式などでダウンロードできます。
2.	AI背景除去（Matting）
機能: アップロードした人物写真などの背景を自動的に削除し、透明な背景（PNG）にします。
技術: 「MODNet」というディープラーニングモデル（ONNX形式）を使用し、高精度なポートレートマッティング（切り抜き）を実現しています。
3.	文字抽出（OCR）
機能: 画像内の文字を読み取り、テキストデータとして抽出します。結果画像には認識された文字の場所（バウンディングボックス）が描画されます。
技術: 「EasyOCR」ライブラリを使用しており、中国語と英語に対応しています。
4.	画像編集ツール（クロップ・リサイズ）
クロップ: スライダー操作で画像の切り抜き範囲（％）を指定し、サーバー側で画像を加工します。
リサイズ: 画像の解像度を「倍率指定」または「ピクセル指定」で変更し、フォーマット（PNG/JPEG）や画質を変換してダウンロードできます。
5.	ユーザー認証システム
機能: ユーザー登録とログイン機能を提供します。
技術: JWT（JSON Web Token）を使用したトークンベース認証を採用しており、各APIエンドポイントは認証済みユーザーのみがアクセスできるように保護されています。データはSQLiteデータベースに保存されます。

技術スタック
フロントエンド:
Streamlit: ユーザーインターフェースの構築。画像のアップロード、プレビュー、各種パラメータの設定、APIとの通信を担当します。
バックエンド:
FastAPI: 高速なREST APIサーバー。
SQLAlchemy / SQLite: ユーザー情報の管理。
JWT: セキュリティと認証。
AI・画像処理ライブラリ:
Dashscope SDK: 画像生成AIの呼び出し。
ONNX Runtime: 背景除去モデル（MODNet）の推論実行。
EasyOCR: 光学文字認識。
Pillow (PIL) / NumPy: 画像の基本的な読み込みと加工。

まとめ
このプロジェクトは、最新のAI技術（生成AI、コンピュータビジョン）を、使いやすいWebインターフェース（Streamlit）を通じて手軽に利用できるように統合したフルスタックアプリケーションの雛形と言えます。
